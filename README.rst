pylw
======
Python Lightweight Webframework.

|Build Status|

This is a somewhat useful web framework.

It's meant to be fast, and to do very little.

Unicode support...none.
~~~~~~~~~~~~
We also don't support python3 at this time.

Complexity...zero.
